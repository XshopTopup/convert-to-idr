1. Ambil Daftar Mata Uang

Gunakan endpoint ini untuk mendapatkan semua kode kurs dan nama mata uang yang didukung.

GET /api/convert
fetch('/api/convert')
  .then(res => res.json())
  .then(json => {
    console.log(json.data); // Array objek: {kode, name, val}
  });
<?php
$res = file_get_contents("https://your-app.vercel.app/api/convert");
$json = json_decode($res, true);
print_r($json['data']);
?>

2. Konversi Mata Uang

Endpoint untuk menghitung nominal mata uang asing ke Rupiah (IDR).

GET /api/convert?val=[nominal]&kurs=[kode]
fetch('/api/convert?val=10&kurs=usd')
  .then(res => res.json())
  .then(data => {
    console.log(`Hasil: ${data.result}`);
  });
<?php
$url = "https://your-app.vercel.app/api/convert?val=10&kurs=usd";
$data = json_decode(file_get_contents($url), true);
echo "Rp " . number_format($data['result'], 0, ',', '.');
?>
import requests
res = requests.get("https://your-app.vercel.app/api/convert?val=10&kurs=usd")
print(res.json()['result'])
